##steps for starter Template 

### 1. clone the repository 
```bash
git clone https://github.com/HJ-Chinthan7/backend-typescript.git  <Project Name>
```

### 2. Move in to the folder structurecd 
```bash
cd <Project name>
```
### 3.install npm dependencies
```bash
npm i
```

### 4.create a new file in the root directory as .env and add port value 
```bash
cd .env
```
```env
PORT=
```
### 5.start the server 
```bash
npm run dev
```
